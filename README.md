# personal-website

Ale Arbizzani's personal website. Plain HTML/CSS/JS, no build step, deployed via GitHub Pages.

## Local preview

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000.

## Deployment

Pushes to `main` are served automatically by GitHub Pages at:
https://alearbizzani.github.io/personal-website/
